PANDUAN LENGKAP TRIGONOMETRI

Materi Persiapan Ulangan, TKA, dan UTBK

1. Pengenalan Sudut dan Kuadran

Trigonometri berfokus pada hubungan antara sudut dan sisi segitiga. Dalam koordinat Kartesius, satu lingkaran penuh (360°) dibagi menjadi empat Kuadran. Pemahaman tentang kuadran sangat krusial karena menentukan nilai positif atau negatif dari fungsi sinus, kosinus, dan tangen.

Jembatan Keledai: "SEMUA SINdiket TANgannya KOSong"
  • Kuadran I (0-90°): SEMUA (Sin, Cos, Tan) bernilai POSITIF.
  • Kuadran II (90-180°): Hanya SIN yang POSITIF.
  • Kuadran III (180-270°): Hanya TAN yang POSITIF.
  • Kuadran IV (270-360°): Hanya COS yang POSITIF.

2. Sudut Istimewa: Jantung Trigonometri

Sudut istimewa adalah sudut-sudut yang nilai trigonometrinya dapat ditentukan secara eksak tanpa kalkulator. Sudut ini meliputi 0°, 30°, 45°, 60°, dan 90°. Kamu wajib menghafal nilai-nilai ini karena sering muncul di soal TKA dan ujian nasional.

Perhatikan pola pada Sinus: nilainya naik dari 0 ke 1 (0, 1/2, 1/2√2, 1/2√3, 1). Kosinus adalah kebalikannya, nilainya turun dari 1 ke 0.

3. Relasi Sudut (Sudut Berelasi)

Bagaimana cara menghitung sin 150°? Kita menggunakan relasi sudut dengan acuan 180° atau 360° agar fungsi tidak berubah.

Rumus Utama:
  • K-II (180 - α): sin(180-α) = sin α
  • K-III (180 + α): tan(180+α) = tan α
  • K-IV (360 - α): cos(360-α) = cos α

Contoh: sin 150° = sin(180 - 30) = sin 30° = 1/2.

4. Identitas Dasar & Aturan Segitiga

Identitas Pythagoras: sin²x + cos²x = 1. Identitas ini adalah alat paling sakti untuk mencari nilai fungsi lain jika salah satu fungsi diketahui. Selain itu, pahami aturan Sinus dan Kosinus untuk menyelesaikan masalah pada segitiga sembarang.

5. Implementasi Pemrograman (Cross-Language)

Bagaimana komputer menghitung trigonometri? Berikut adalah contoh implementasi dalam bahasa Java dan C++ untuk menghitung Sinus menggunakan radian.

// Java Implementation
public class TrigCalc {
  public static void main(String[] args) {
    double degrees = 30.0;
    double radians = Math.toRadians(degrees);
    System.out.println("Sin(30) = " + Math.sin(radians));
  }
}
// C++ Implementation
#include <iostream>
#include <cmath>

int main() {
  double deg = 45.0;
  double rad = deg * M_PI / 180.0;
  std::cout << "Cos(45) = " << std::cos(rad) << std::endl;
  return 0;
}
